## Description
TweenLabs is a premium, open-source library of copy-paste GSAP (GreenSock Animation Platform) animation components built for React 19 and Next.js 16. Every component is self-contained, production-ready, and installable via CLI.

## Tech Stack
- Framework: Next.js 16 (App Router)
- UI Library: React 19
- Animation: GSAP 3.15 + @gsap/react
- Styling: Tailwind CSS 4
- Scrolling: Lenis 1.3
- Language: TypeScript (strict mode)
